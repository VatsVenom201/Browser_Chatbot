How to Use

🔍 Analyze Page

Click this first! It reads the current tab and prepares the AI to answer questions about it.

📝 Summarize Page

Gets a quick bulleted summary of everything on the current webpage.

✂️ Summarize Selected

Highlight any text on the page first, then click this to summarize just that part.

💬 Chat

Type any question at the bottom. The AI uses the context of the page you analyzed to provide accurate answers.
